# cdesign project

This project was generated by the cdesign skill. Follow these rules when editing.

## Edit Mode

All changes must be applied as DELTA — never rewrite the project from scratch.

### Preserve always:
- Visual metaphor and vibe direction
- Motion hierarchy: hero > section transitions > UI hover > ambient
- Typography system (do NOT change fonts)
- Device tier fallbacks (DeviceTierProvider)
- Lenis + GSAP ticker sync (single RAF clock)
- Anti-slop rules

### Do NOT:
- Replace scroll/shader animations with simple CSS fade-ins
- Simplify the hero scene unless explicitly asked
- Remove animations unless explicitly asked
- Use `transition: all` (list specific properties)
- Stack multiple heavy effects in one viewport
- Animate width/height/top/left/box-shadow/filter
- Add `key={index}` in lists
- Use `h-screen` (use `min-h-[100dvh]`)
- Use `useState` for mousemove (use MotionValue)

### Temporal discipline:
- Hover feedback: 120–220ms
- UI transitions: 300–500ms
- Scene transitions: 800–1400ms
- Ambient motion: 3–12s
